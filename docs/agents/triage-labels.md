# Triage Labels

This project uses the canonical Matt Pocock triage roles as local markdown status names.

## Category roles

| Role | Label |
| --- | --- |
| `bug` | `bug` |
| `enhancement` | `enhancement` |

## State roles

| Role | Label |
| --- | --- |
| `needs-triage` | `needs-triage` |
| `needs-info` | `needs-info` |
| `ready-for-agent` | `ready-for-agent` |
| `ready-for-human` | `ready-for-human` |
| `wontfix` | `wontfix` |

Each triaged issue should have exactly one category role and one state role.
