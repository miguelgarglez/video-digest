# Guided Terminal UI Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make `video-digest` without arguments launch a low-density guided TUI covering creation, Library, Settings, setup, and diagnostics.

**Architecture:** Model navigation and effects as a renderer-independent state machine. An `@opentui/core` adapter renders screens and translates key events; existing CLI application services remain the only route to Ingestion, Transcript, setup, credentials, and the Artifact Library.

**Tech Stack:** Bun, TypeScript, Bun test, `@opentui/core` after explicit user approval.
